## Instructions

1. Install pymongo
```
pip install pymongo
```
2. Restore the Chat database
```
mongorestore --db Chat path_to_Chat_folder
```
3. Start the server

```
python3 main.py
```
4. Open any web browser and connect to http://localhost:7070
5. Make new friends and have fun!
